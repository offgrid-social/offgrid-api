# offgrid-api

Reference backend implementation of the offgrid protocol.

This implementation exists for testing, experimentation, and interoperability.
It is **not** a mandatory or canonical backend.

## Scope
- Protocol implementation
- API endpoints
- Validation logic
- Federation handling

## Non-Goals
- No vendor lock-in
- No assumption that this is the only correct backend

Other implementations are encouraged.
